Architecture

How it works

A clean RAG pipeline from document ingestion to accurate AI response in four steps.

1

Document Ingestion

Your content (PDFs, HTML, product specs, FAQs) is parsed and split into semantic chunks optimised for retrieval.

2

Vector Embedding

Each chunk is converted into a vector embedding and stored. This powers semantic search based on meaning, not keywords.

3

Retrieval

When a user asks a question, the pipeline finds the most relevant chunks from your data using similarity search.

4

Generation via Claude API

The retrieved context is passed to Claude alongside the user's question. Claude generates a grounded, accurate response every time.
